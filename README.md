# Development of a Cyber Physical System in Real Time Bolt Defect Detection using Machine Learning based Automated Quality Control

## 📌 Project Overview

An AI-powered cyber-physical system for real-time bolt defect detection that integrates YOLOv8-based computer vision with a Flask web interface and Arduino-driven hardware actuation. The system enables automated, high-speed quality control by instantly classifying bolts and triggering physical responses for defective components. Designed for Industry 4.0, it ensures efficient, accurate, and scalable inspection in modern manufacturing environments.
